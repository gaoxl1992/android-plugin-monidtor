# 更新日志

## [1.0.0] - 2024-12-17

### 新增功能
- ✨ 首次发布
- ✨ 支持监听 Android 系统所有应用的通知消息
- ✨ 提供通知监听权限检查接口 `checkPermission()`
- ✨ 提供打开系统设置页面接口 `openSettings()`
- ✨ 提供开始监听接口 `startListening(callback)`
- ✨ 提供停止监听接口 `stopListening()`
- ✨ 提供服务状态检查接口 `isServiceRunning()`
- ✨ 支持获取通知的应用包名、标题、内容、时间戳
- ✨ 自动处理生命周期，防止内存泄漏

### 文档
- 📝 完整的 README.md 使用说明
- 📝 快速开始指南
- 📝 插件市场描述文档
- 📝 详细的 API 文档和示例代码
- 📝 常见问题解答

### 技术特性
- 🔧 基于 Android NotificationListenerService
- 🔧 支持 Android 5.0（API 21）及以上
- 🔧 使用 uni-app 原生插件开发规范
- 🔧 提供 AAR 打包格式
